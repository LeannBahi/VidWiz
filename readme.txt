Overview:
---------
VidWiz is an interactive tool that generates a polished video with a soundtrack
from selected video. In order to start the program, the user needs to download
some python modules and files, then runs “python VidWiz.py”.


User flow:
----------
VidWiz will first ask the user to select videos from their hard drive. 
The end product is a saved video montage of the selected video with a generated
soundtrack.


Installation guide:
-------------------


You'll need Python 2.7 in order to run this program.
VidWiz relies on the following libraries to perform video analysis.


1/ OpenCV :
download OpenCV from the following link
http://sourceforge.net/projects/opencvlibrary/files/
Install OpenCV onto your computer.


note: There’s a great step-by-step instruction for MacOs here:
http://www.mobileway.net/2015/02/14/install-opencv-for-python-on-mac-os-x/


2/ Numpy
download numpy from the following link 
http://sourceforge.net/projects/numpy/files/
Install numpy onto your computer




3/ Python Imaging Library (PIL)
download PIL from the following link 
https://pypi.python.org/pypi/Pillow
Install PIL onto your computer
